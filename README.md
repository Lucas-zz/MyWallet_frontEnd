## How to run

1. Clone this repository

2. Install dependencies
```bash
npm i
```

3. Run the app with
```bash
npm run start
```

4. Finally, access http://localhost:TO_BE_DEFINED on your favorite browser (unless it's Internet Explorer. In this case, review your life decisions)
